# AI-Powered Predictive Maintenance System for IoT Devices

## Overview
This project is an AI-driven industrial predictive maintenance system that uses IoT sensor data simulation and Machine Learning to predict machine failures before they occur.

It demonstrates a complete end-to-end ML pipeline including data generation, preprocessing, model training, evaluation, visualization, and real-time prediction.

---

## Problem Statement
In industries such as manufacturing, aviation, and power plants, unexpected machine failures cause:

- High maintenance costs
- Production downtime
- Safety risks

This project solves this by predicting failures in advance using sensor data.

---

## Solution
We built a machine learning system that:

- Simulates IoT sensor data
- Learns failure patterns
- Predicts machine breakdown risk
- Visualizes system health

---

## Dataset
Synthetic IoT sensor dataset includes:

- Temperature
- Vibration
- Pressure
- RPM
- Wear Level
- Failure Label (0/1)

Data Generation → Preprocessing → Feature Engineering → ML Model → Prediction → Visualization
